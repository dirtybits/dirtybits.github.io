ABOUT CONCEAL

{ What is Conceal? }

Conceal is a privacy protected digital cash system. It is completely decentralized, without the need for a central server or trusted parties. Users hold the crypto keys to their own money and transact directly with each other over a secure peer-to-peer network. Conceal is an open source scalability-focused, untraceable, secure, and fungible cryptocurrency using CryptoNote (Masari's CN Fast variant). The decentralised network supports encrypted messages and deposits with interest.

FEATURES

{ Private }

Conceal uses ring signatures and one-time addresses for truly anonymous payments.

{ Untraceable }

Conceal's transactions can't be linked between the sender and the recipient.

{ Decentralised }

Conceal follows Satoshi Nakamoto's original vision of decentralised, trustless cryptocurrency, i.e. a secure digital cash operated by a network of users. Transactions are confirmed by distributed consensus, and then recorded on the blockchain immutably. Third-parties do not need to be trusted to keep your money safe

{ Fungible }

Conceal is truly fungible, thanks to built-in privacy features. Just like cash, all coins are equal, changeable. It is extremely unlikely that a coin will ever be blacklisted by any party due to its association in previous transactions

Mining

{ Block Rewards }

Conceal's block reward model uses an innovate approach. The block reward starts small and get bigger with time as the blockchain network grows, so does the cost of mining. At the beginning every network have a small number of transactions per day and the blockchain is still a small file but with time the size will go up and the effort to keep it running will be bigger. More miners will be needed to keep it running sound and safe. To reward properly this mining effort we will start with 5 CCX per block and raise it by 0.25 CCX every month until we reach 20 CCX per block (capped - max reward in 5 years).
